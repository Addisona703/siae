# siae-auth

### 1. AuthService
1. 用户登录
2. 用户注册
3. 刷新令牌
4. 登出

### 2. PermissionService
1. 添加权限
2. 查询权限
   1. 查询权限列表
   2. 查询权限详情（通过ID）
3. 更新权限
4. 删除权限

### 3. RoleService
1. 创建角色
2. 查询角色
   1. 查询角色列表
3. 更新角色
4. 删除角色
5. 给角色添加权限
6. 查询角色权限
7. 修改角色权限
8. 删除角色权限

### 4. UserRoleService
1. 给用户添加角色
2. 查询用户角色
3. 修改用户角色
4. 删除用户角色

### 5. UserPermissionService
1. 给用户添加权限
2. 查询用户权限
3. 修改用户权限
4. 删除用户权限

### 6. LogService
1. 创建登录日志
2. 查询登录日志
